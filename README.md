# Fanavrd data mining contest 2018
### Install dependencies:
```shell
$ pip install -r requirements.txt
```
### Train the model:
```shell
$ python train.py <item|all> # items are : A,B,...
```

### Predict with the model:
```shell
$ python main.py <item> <data-seperated-by-space> # items are : A,B,...
```
